# Tavern UI Localization

Use this reference when adding or modifying tavern UI language support.

Files:

- `/opt/data/apps/tavern-runtime/web/i18n.js`: main UI strings.
- `/opt/data/apps/tavern-runtime/server.py`: actor-card intimacy labels in `INTIMACY_LEVEL_I18N` and `INTIMACY_BLURB_I18N`.

Rules:

- Use the `en` object as the full key template. Add every key for the new locale.
- Preserve interpolation placeholders such as `{x}` exactly.
- Do not translate brand anchors such as `Liveware` or the `✦` mark.
- Language choice is driven by ClawChat/liveware `?lang=`; do not add a manual language switch unless the product requirement changes.
- After editing, restart with `sh /opt/data/skills/creative/tavern/scripts/bringup.sh` and reload the liveware app.
